# Iris Classification Model Comparison

This project compares several machine learning classification models using the Iris dataset from scikit-learn.

## 📌 Overview

The goal of this project is to classify Iris flower species based on petal length and petal width. Several classification algorithms are trained and visualized to compare how each model separates the Iris classes.

## ✨ Features

- Load Iris dataset from scikit-learn
- Explore dataset structure and missing values
- Split data into training and testing sets
- Apply feature standardization using StandardScaler
- Train multiple classification models
- Visualize decision boundaries
- Compare model behavior using different algorithms
- Analyze regularization effect in Logistic Regression
- Save trained Random Forest model using Pickle

## 🧠 Models Used

- Perceptron
- Logistic Regression
- Support Vector Machine / SVM
- Random Forest Classifier
- K-Nearest Neighbors / KNN

## 🛠️ Tech Stack

- Python
- Jupyter Notebook
- NumPy
- Pandas
- Matplotlib
- scikit-learn
- Pickle

## 📁 Dataset

This project uses the Iris dataset from scikit-learn. The selected features are:

- Petal length
- Petal width

The target classes are:

- Iris Setosa
- Iris Versicolor
- Iris Virginica
